# Donnees locales

Ce dossier est volontairement ignore par Git pour eviter de publier des donnees brutes ou des fichiers lourds.

- `raw/` : fichiers originaux `DATA*.TXT` / `DATA*.DAT`.
- `simulated/` : donnees simulees.
- `processed/` : exports intermediaires ou nettoyes.

Les sous-dossiers sont conserves dans le depot avec des fichiers `.gitkeep`.
